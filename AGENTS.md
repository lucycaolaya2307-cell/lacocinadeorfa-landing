# Project Instructions

- For this project only, whenever an instruction involves changing, replacing, adding, or using images, convert the image asset to WebP before using it in the site.
- Apply this automatically for image changes unless the user explicitly asks for a different format.
- If a source image or product cannot be confidently identified by name, do not assign it to a named product card. Use it only as collage, gallery, or ambient website imagery.
- Whenever the user confirms that a PR was merged, switch to `main` and run `git pull --ff-only` before starting the next change.
- From now on for this project, work directly on `main` and push changes to `main`; do not create PRs unless the user explicitly asks for one.
